Descripción del Prototipo

"Prototipo Inalámbrico de Monitoreo de Calidad del Agua de Canales de Riego Usando Tecnología LoRa y 2G"

Este prototipo es de monitoreo IoT diseñado para medir en tiempo real la calidad del agua de riego en entornos agrícolas. Utiliza una placa LoRa 32 como unidad sensora y un módulo SIM7000 para el envío de datos mediante la red 2G, garantizando conectividad incluso en zonas rurales sin acceso a internet tradicional. La información recolectada se transmite a la nube, donde se almacena en InfluxDB y se visualiza mediante dashboards.

Las principales características del prototipo incluyen:

  • Medición en tiempo real de parámetros como pH, turbidez y temperatura.
  • Transmisión autónoma de datos solo cuando hay nuevas lecturas.
  • Energía solar como fuente principal de alimentación.
  • Almacenamiento en la nube (InfluxDB) para registros históricos.
  • Visualización web interactiva mediante dashboards.
  • Instalación sencilla y operación sin conocimientos técnicos.

El prototipo es ideal para aplicaciones de monitoreo ambiental, domótica, agricultura inteligente, y cualquier escenario que requiera la recolección y análisis de datos de sensores.

Instructivo de Uso

Pasos para Usar el Prototipo

Este instructivo te guiará a través de los pasos necesarios para configurar y utilizar tu prototipo de monitoreo de calidad de agua. Asegúrate de seguir cada paso cuidadosamente para garantizar un funcionamiento óptimo.

  1. Colocación:

    Poner el prototipo en un lugar estable, donde el agua cubra correctamente los sensores y donde le dé el sol. Asegúrate de que no haya obstáculos que puedan interferir con la señal.

  2. Activación del Transmisor:

    Ver que prenda la pantalla donde aparezca "LoRa transmisora: Datos enviados". Esto indica que el módulo está funcionando correctamente y enviando datos.

  3. Activación del Receptor:

    Encender la otra LoRa receptora y visualizar la pantalla donde aparezca "LoRa receptora: Datos recibidos". Esto confirmará que los datos están siendo recibidos correctamente.

  4. Encender LilyGO:

    Prender la LilyGO y esperar a que conecte a la red durante aproximadamente 10 minutos. Asegúrate de que el SIM tenga crédito suficiente para enviar datos a la nube.

  5. Verificación del Dashboard:

    Accede al dashboard en la página web proporcionada. Verifica que todos los parámetros se estén actualizando correctamente. Esto te permitirá monitorear la calidad del agua en tiempo real.

  6. Mantenimiento Regular:

    Realiza chequeos periódicos del sistema para asegurarte de que los sensores estén limpios y funcionando correctamente. Reemplaza cualquier componente que muestre signos de desgaste.

  7. Solución de Problemas:

    Si encuentras problemas, revisa las conexiones y asegúrate de que todos los dispositivos estén encendidos. Consulta el manual del usuario para obtener más información sobre la solución de problemas.

Siguiendo estos pasos, podrás aprovechar al máximo tu prototipo y garantizar un monitoreo efectivo de la calidad del agua de riego.
